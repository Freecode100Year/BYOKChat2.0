🔧 工具与扩展
填入后开启功能栏「🌐 联网」，模型即可自主搜索互联网、在回答下方附参考来源。需自备 Tavily Key（tavily.com 注册，免费层约每月 1000 次）。需模型支持工具调用（主流模型都支持）。
另一种方式无需此项：直接用自带联网的供应商——如 Perplexity 的 sonar 模型，或 OpenRouter 模型名加 :online 后缀。
连接支持 HTTP/SSE 的远程 MCP 服务器，让模型自主调用其工具（需模型支持工具调用）。填地址后点「连接」拉取工具，勾「启用」生效。本地 stdio 类 MCP（npx 启动的）无法连接，仅支持远程 https 端点；需 OAuth 的服务暂不支持。
定义可复用的提示词，输入框里打 / 即可快速选用。例如「翻译」→「把以下内容翻译成地道英文：」。
把一个 Skill 导入成助理（用其指令当系统提示词）或快捷指令。兼容 Skill Builder 生成的文件。含可执行脚本的 Skill 仅指令部分会被导入。
💾 数据管理
「合并」追加备份里的对话与助理（同名助理自动跳过），保留本机现有数据；「覆盖」清空本机、用备份完全替换。默认导出不含 Key，可安全分享。
把你的助理、快捷指令、全局指令、MCP 配置打包成一个可分享的文件（不含 Key / 对话 / 记忆等隐私）。导入时合并到现有配置。适合把一套「智能体方案」分享给别人。
把所有供应商（名称 / 地址 / API Key / 额度 / 模型）导出成易读、可手动编辑的 Markdown，按相同格式可批量导入。⚠️ 文件含明文 Key，请妥善保管。导入时同名供应商会被更新、新供应商会追加。

🔒 隐私与匿名说明

我们尽量少地接触你的数据。以下是真实的数据流向。
  • ✓无需登录：没有账号、没有注册，不用 Cookie 追踪你的身份。
  • ✓本地保存：对话、设置、Key 等数据只保存在当前浏览器的 localStorage + IndexedDB，可随时导出备份或手动清空。
  • ✓Key 不上传：你填写的 API Key 只保存在你的浏览器本地。
  • ✓知识库本地检索：RAG 的文档切片与匹配都在浏览器完成，文档不上传。
  • ○消息/图片/检索片段会发给 AI 服务商：得到回复需转发给你 key 对应的服务商，其处理遵循该服务商隐私政策。
本站托管于 Cloudflare，平台层可能产生标准网络访问日志。清除浏览器数据即可删除本地保存的一切。
